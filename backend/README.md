# Finance Guardian Backend

This is the backend for the AI Personal Finance Guardian for Students project.

## Current Features
- Flask backend setup
- MongoDB database connection
- User authentication (Register & Login)
- JWT-based security (in progress)

## Tech Stack
- Python
- Flask
- MongoDB
- JWT Authentication

## Setup Instructions
1. Clone the repo
2. Create a virtual environment
3. Install dependencies:
pip install -r requirements.txt
4. Start MongoDB service
5. Run the server:

## python app.py status
